# lt-hook
Lithuanian letters without lithuanian keyboard layout

## Usage
When you run the program, hook activates and starts to work everywhere.
When you hold down print screen button, some keys change.

### Key change list
* $ -> € (only uppercase)
* a -> ą
* c -> č
* e -> ę
* w -> ė
* i -> į
* s -> š
* u -> ų
* y -> ū
* z -> ž

## Installation
If you want for this hook to work on startup too, move the shortcut of compiled .exe to Startup folder.
1. Press Win + R
2. Type shell:startup
3. Copy compiled .exe shortcut into opened folder
